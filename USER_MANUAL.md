# HeartEcho 使用说明书

## 1. 简介
HeartEcho 是一款专为您设计的语音留言本。不需要复杂的操作，只需要用特制的"卡片"（NFC标签）碰一下手机(也可以纯触屏操作)，就可以给家人留言，或者听家人给您的留言。

## 2. 家人听了吗？（消息状态）
您可以看手机屏幕的 **右上角**：
- **🔴 红色（未读）**：您发过去的话，家人 **还没来得及听**。
- **🟢 绿色（已读）**：您发过去的话，家人 **已经听过了**，放心吧！

## 3. 认识您的卡片
您应该有三张特殊的卡片（或贴纸）：
- **🔴 录音卡**：用来说话留言。
- **🟢 播放卡**：用来听最新的留言。
- **🛑 停止卡**：用来停止当前的动作（比如不想录了）。

## 4. 怎样给家人留言（录音）

1. **碰一碰**：拿起 **🔴 录音卡**，贴在手机背面的感应区。
2. **听提示**：手机会震动一下，并说话提示您"正在录音，请说话"。
3. **开始说话**：
   - 屏幕背景会变 **绿色**，您可以开始说话了。
   - **声音条说明**：屏幕右侧有一个竖条，会随着您的声音上下跳动。**请确认它在跳动**，说明手机听到了您的声音。
   - **提前结束/取消**：屏幕下方有两个透明按钮（防止误触，需要**长按**）：
     - **停止 (■)**：**长按**右边的方块按钮，直到圆圈转满，表示"说完了"。
     - **取消 (✕)**：**长按**左边的叉号按钮，直到圆圈转满，表示"不要了"。
   - **倒计时**：您一共有 **30秒** 的时间。时间快到时，屏幕会变 **黄色**，最后变 **红色** 并闪烁提醒。
4. **发送留言**：
   - 如果说完了，屏幕会出现两个大按钮。
   - **下半个屏幕（绿色）**：点击这里，把话发给家人（会有"✓"符号）。
   - **上半个屏幕（红色）**：如果不满意，点击这里取消（会有"✕"符号）。
   - *提示：您也可以再次碰一下卡片来结束录音。*

## 5. 怎样听家人的留言（播放）

1. **碰一碰**：拿起 **🟢 播放卡**，贴在手机背面的感应区。
2. **听留言**：
   - 手机会说话提示"正在为您播报最新的留言"。
   - 接着就会播放家人给您的录音。
   - 屏幕上会有进度条显示播放到哪里了。
3. **屏幕上的按钮（也可以用手点）**：
   - **中间的大按钮 (▶/⏸)**：暂停或继续播放。
   - **左边的蓝色按钮 (⏮)**：从头开始重听这一条。
   - **右边的红色按钮 (■)**：停止播放。
   - **进度条**：您可以按住屏幕中间的横条拖动，想听哪里听哪里。
4. **重听一遍**：如果没听清，除了点蓝色按钮，也可以再用卡片碰一下手机，也会从头开始放。

## 6. 没有卡片也可以用（直接点屏幕）

如果您身边没有卡片，或者不想用卡片，也可以直接点击手机屏幕上的大按钮：

- **我要录音**：点击屏幕中间最大最圆的 **🟢 绿色按钮**（写着"开始录音"）。这和用 **🔴 录音卡** 是一样的。
- **我要听留言**：点击下方的 **🟡 黄色长条按钮**（写着"听取录音"）。这和用 **🟢 播放卡** 是一样的。
- **找以前的留言**：点击上方的 **🟣 紫色长条按钮**（写着"历史对话"），可以看到以前所有的录音记录。

## 7. 常见问题

**Q: 录音的时候不想录了怎么办？**
A: 您可以用 **🛑 停止卡** 碰一下手机，或者等待30秒自动结束，然后在屏幕上点击上半部分的 **红色区域**（取消）。

**Q: 手机没反应怎么办？**
A: 请把卡片贴在手机背面中间的位置，稍微停留一下，不要移开得太快。

**Q: 屏幕一直闪红色是什么意思？**
A: 这是在提醒您录音时间马上就要结束了（还剩最后5秒），请尽快说完。
